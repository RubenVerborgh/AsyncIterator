Class: BufferedIterator

asynciterator~ BufferedIterator

An iterator that maintains an internal buffer of items. This class serves as a base class for other iterators with a typically complex item generation process.

maxBufferSizenumber

The maximum number of items to preload in the internal buffer. A BufferedIterator tries to fill its buffer as far as possible. Set to Infinity to fully drain the source.
